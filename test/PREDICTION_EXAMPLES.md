# 대시보드 예측 입력값 예시 모음

훈련 데이터를 기반으로 한 다양한 입력값 조합과 예상 결과입니다.

## 📊 실제 훈련 데이터 기반 예시

### 🟢 양호한 결과 예상 (190 mAh/g 이상)

#### 예시 1: 높은 온도 + 중간 시간
```
소성온도: 950
소성시간: 18
예상 결과: 약 484 (더미 모델 기준)
실제 데이터 유사: 951.5°C, 19.2h → 194.94 mAh/g
```

#### 예시 2: 매우 높은 온도 + 긴 시간
```
소성온도: 980
소성시간: 20
예상 결과: 약 500
실제 데이터 유사: 978.6°C, 15.3h → 190.02 mAh/g
```

#### 예시 3: 중간 온도 + 중간 시간
```
소성온도: 900
소성시간: 15
예상 결과: 약 457.5
실제 데이터 유사: 890.3°C, 12.2h → 205.95 mAh/g
```

#### 예시 4: 높은 온도 + 짧은 시간
```
소성온도: 970
소성시간: 10
예상 결과: 약 490
실제 데이터 유사: 973.0°C, 9.4h → 189.45 mAh/g
```

### 🟡 경계선 근처 (190 mAh/g 근처)

#### 예시 6: 낮은 온도 + 긴 시간

## 📈 실제 훈련 데이터에서 가져온 정확한 값들

### 높은 방전용량 (200+ mAh/g)
```

2. 소성온도: 890.3, 소성시간: 12.2 → 방전용량: 205.95
```

## ⚠️ 주의사항

현재 시스템은 **더미 모델**을 사용 중입니다:
- 예측 공식: `(소성온도 + 소성시간) / 2`
- 실제 ML 모델이 아닌 간단한 계산식입니다
- 실제 모델 통합 시 더 정확한 예측이 가능합니다
