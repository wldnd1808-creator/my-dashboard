# 예측 입력 가이드

## 올바른 입력값 범위

훈련 데이터를 기반으로 한 올바른 입력 범위:

- **소성온도 (Feature1)**: 750 ~ 1000 °C
- **소성시간 (Feature2)**: 8 ~ 24 시간

## 추천 입력값 예시

### 예시 1: 양호한 예측값 예상
- 소성온도: **900** °C
- 소성시간: **15** 시간
- 예상 방전용량: 약 190~195 mAh/g

### 예시 2: 높은 방전용량 예상
- 소성온도: **950** °C
- 소성시간: **18** 시간
- 예상 방전용량: 약 190~200 mAh/g

### 예시 3: 낮은 방전용량 예상 (불량 가능)
- 소성온도: **800** °C
- 소성시간: **10** 시간
- 예상 방전용량: 약 185~190 mAh/g (품질 기준선 근처)

## 잘못된 입력값 예시

❌ **피해야 할 값:**
- 소성온도: 3.6 (너무 낮음)
- 소성시간: 4.3 (너무 낮음)
- → 이런 값은 훈련 데이터 범위를 벗어나서 예측이 부정확합니다.

## 품질 판정 기준

- **양호**: 방전용량 ≥ 190 mAh/g
- **불량**: 방전용량 < 190 mAh/g (빨간색으로 표시됨)

## 사용 방법

1. 대시보드의 "예측 실행" 섹션으로 이동
2. 소성온도 입력 (750~1000 사이의 값)
3. 소성시간 입력 (8~24 사이의 값)
4. "예측" 버튼 클릭
5. 결과 확인:
   - 예측값과 품질 판정 표시
   - 예측 결과 테이블에 자동 추가
   - 차트 자동 업데이트
